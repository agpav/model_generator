# model-generator

Generate java model from any source input file.

## Usage

## Tool
Use at compile time

#### Input
Grammar file - This will be use to parse the input file
input file - input data file

#### Output
Java class - Source Java file

## Library
Use at runtime

#### Input
Grammar file - This will be use to parse the input file
input file - input data file

#### Output
Java object - Runtime java object
